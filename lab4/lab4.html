Introduction to CSS

CSS (Cascading Style Sheets) allows you to create rules that specify how the content of an element should appear. Only one elelment can be named with an "id". ID is identified by using a hash-tag (#). Justify means to evenly distribute from left to right. The width of the border is set to zero pixels by default. The border can be set to have specific elements. The border has three properties: the assigned name, width and the color. Add a margin to create a space betwween boxes. A margin is the distance of the outside element. Padding is the distance inside the box.
